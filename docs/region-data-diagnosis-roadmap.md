# Region Data Diagnosis Roadmap

## 1. V1

- Region is used as context only.
- The diagnosis is rule-based.
- AteFlo does not perform real-time crawling.
- AteFlo does not perform competitor analysis.
- AteFlo does not claim local ranking analysis.
- Region wording can make recommendations feel more natural, but it must not imply live market data.

## 2. V2

- AI API can classify free text and generate more personalized module output inside AteFlo.
- Region plus business type can influence recommendation order and wording.
- Saved business profiles can improve repeated use if an account system exists.
- No ranking, sales, lead, booking, or revenue guarantees.

## 3. V3

- Approved external APIs and allowed data sources can be connected later.
- Possible integrations include Naver local/search/trend data, local business category signals, regional keyword signals, and channel priority suggestions.
- No unauthorized scraping.
- Comply with API terms and local platform rules.
- Do not guarantee rankings, sales, leads, customers, bookings, income, or business growth.
