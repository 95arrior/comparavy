# Homepage Builder Module Blueprint

This document defines the first real paid module for **AI 온라인 영업 세팅
키트**.

Module name:

홈페이지 만들기

## User-Facing Promise

가게나 서비스를 처음 보는 사람이 바로 이해하고 문의할 수 있도록 홈페이지 첫
화면과 기본 문구를 순서대로 정리합니다.

## What This Module Helps Users Create

- 홈페이지 첫 문장
- 서비스 소개문
- 문의/예약 CTA
- FAQ
- SEO 제목/설명
- 도메인·배포 체크리스트

## Target Users

- 가게/매장 운영자
- 로컬 서비스업
- 예약 기반 서비스
- 온라인 상담/교육 서비스
- 홈페이지가 없거나 첫 화면 문구가 약한 사람

## User Pain

- 홈페이지에 뭐라고 써야 할지 모름
- 방문자가 뭘 하는 곳인지 바로 이해하지 못함
- 문의/예약 버튼 문구가 약함
- 검색에 보일 제목과 설명을 어떻게 써야 할지 모름
- 도메인/배포/분석 세팅이 복잡함

## Paid Module Flow

This flow is paid content. Until payment/access control exists, the full prompts,
templates, detailed checklists, and complete outputs must not be exposed
publicly.

### 1. 사업 정보 확인

Questions:

- 어떤 일을 하세요?
- 누구에게 보여주고 싶으세요?
- 가장 알리고 싶은 서비스는 무엇인가요?

### 2. 첫 화면 문구 만들기

Output:

- 한 줄 설명
- 짧은 소개 문장
- 신뢰 문구

### 3. 서비스 소개문 만들기

Output:

- 대표 서비스 설명
- 고객이 얻는 결과
- 주의할 표현 체크리스트

### 4. 문의/예약 CTA 만들기

Output:

- 버튼 문구
- 상담/예약 유도 문구
- 과장 없는 안내문

### 5. FAQ 만들기

Output:

- 처음 방문자가 궁금해할 질문
- 답변 구조
- 문의 전환용 FAQ

### 6. SEO 제목/설명 만들기

Output:

- 검색용 제목 후보
- 설명 문구 후보
- 과장 표현 체크리스트

### 7. 도메인·배포 체크리스트

Output:

- 도메인 준비
- 페이지 배포 준비
- GA4/Search Console 확인
- 네이버 서치어드바이저 선택 점검

Important:

This is guidance/checklist in V1. Do not claim automatic domain purchase,
deployment, analytics installation, or SEO ranking.

## V1 / V2 / V3

### V1

- guided questions
- ready-to-copy prompts/templates
- checklist
- no AI API
- no automatic deployment
- no payment automation

### V2

- AI-generated copy inside AteFlo
- regenerate / tone change / copy
- optional download/export
- saved business profile if account system exists

### V3

- landing page generation
- export to HTML/Notion/PDF
- deployment guidance
- domain checklist
- analytics setup guidance
- optional integration with payment/domain tools if technically and legally safe

## Public Preview Content

Public preview content can show the shape and value of the module without
giving away the full paid system.

Allowed public copy:

- This module helps you build the first screen of your website.
- You will be able to create headline, service copy, CTA, FAQ, SEO
  title/description, and setup checklist.
- Tiny example:
  "성수동에서 웨딩 네일과 손톱 케어를 편안하게 받을 수 있는 네일샵"

Locked copy:

- Full prompts are included after access.
- Full checklist is included after access.
- Full step-by-step flow is included after access.

## Public Preview Must Not Include

- full prompt sequences
- full templates
- detailed SEO/domain/deployment checklist
- complete homepage copy package
- generated final copy beyond the tiny sample

## Analytics

Allowed public analytics params:

- `kit_slug`: `online-sales-setup-kit`
- `module_slug`: `homepage-builder`
- `source_page`
- `action_location`

Never send:

- business name
- location
- service name
- raw answers
- free text
