# AteFlo Content Strategy

AteFlo should publish fewer, stronger shortcuts until the quality gate proves
reliable. The site should not scale public output by repeating the same prompt
template across similar topics.

## Future Daily Mix

Default daily mix after the quality gate is stable:

- 1 Work/Productivity shortcut
- 1 Money/Business/Selling shortcut
- 1 Fun/Personal/Desire shortcut

This creates a balanced public library without overloading one topic cluster.

## Optional 4th or 5th Shortcut

Publish a fourth or fifth shortcut only when all of these are true:

- Quality gates are stable.
- Topics are distinct from that day's other shortcuts.
- Prompt workflows are truly strong.
- Works with tools are topic-specific.
- The shortcut has a clear finished output and review checklist.

Do not default to 5 per day.

## Topic Mix Rules

- Avoid publishing same-topic how-to and tool-decision pairs too early.
- Avoid publishing several shortcuts that differ only by noun swaps.
- Rotate across practical work, business/money, and personal/desire topics.
- Prefer topics with clear search demand, weak SERP coverage, and a concrete
  prompt-builder opportunity.

## Public Example Standard

Every published shortcut should be good enough to show a new user what AteFlo is
for:

- The first screen points toward action.
- The prompt builder is easy to use without explanation.
- The example output matches the promised result.
- The Check before using section catches real topic-specific risks.
- Tool recommendations support the workflow instead of turning the page into a
  generic listicle.
