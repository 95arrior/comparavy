# AteFlo AI Workflow Kit Store

AteFlo is a premium AI Workflow Kit Store and kit studio for real-world
outputs.

Users bring business details, job details, profile notes, or content ideas.
AteFlo packages those inputs into structured AI workflow kits:

Input worksheet -> prompt sequence -> example output -> review checklist -> action plan.

## Site Identity

AteFlo should be positioned as:

- a paid AI workflow kit store
- a kit studio for real-world outputs
- a productized AI work system
- a structured path from user input to usable result

AteFlo should not be positioned as:

- an AI shortcut directory
- a free prompt site
- a generic AI tools directory
- a prompt pack store

## Core Message

Users should not pay for prompt lists. They should pay for packaged workflows
that help them finish valuable tasks with less guessing and safer review steps.

## Factory Metaphor

The factory/studio metaphor is functional, not decorative:

- Inputs go in.
- Workflow modules assemble the kit.
- Sample outputs come out.
- The full kit unlocks the complete workflow.

Use existing AteFlo cards, pills, borders, teal accents, and light motion. Do
not add cartoon factory illustrations, image assets, neon effects, fake
scarcity, or aggressive sales design.

## Homepage Rules

The homepage must feel like a paid product store within five seconds.

Above the fold should communicate:

- AteFlo sells AI workflow kits.
- AI 온라인 영업 세팅 키트 is the flagship product.
- Users can build a small sample kit.
- Full kits include templates, examples, checklists, and action plans.

Shortcut search, featured shortcuts, and free prompt language must not be the
main public experience.

## Sample Kit Builder Rules

The sample builder can generate deterministic template output locally.

It must not:

- call an AI API
- store user input
- send raw user input to analytics
- send input to external services

Allowed analytics params:

- kit_slug
- source_page
- action_location
- preview_field_count
- has_preview_generated

## Safety Rules

Factory store pages must not claim or imply:

- guaranteed sales, leads, rankings, customers, income, hiring, dating success,
  or business growth
- fake testimonials
- fake purchase counts
- fake urgency
- fake discounts
- fake reviews or review manipulation

AI 온라인 영업 세팅 키트 must present outputs as review-ready drafts,
templates, and checklists, not automatic publishing, automatic installation, or
business growth automation.
